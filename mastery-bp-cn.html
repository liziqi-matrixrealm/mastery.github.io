概述

掌控是基于6轴IMU技术的低成本手套式3D手部交互控制器，通过专有硬件设计与算法优化，实现高精度、低延迟的手部动作捕捉体验

手套形态为触觉反馈系统与触觉采集系统提供了理想载体，具备自校准、捏合检测、高稳定性等核心特性，为自然直观的交互体验奠定基础

产品可广泛应用于具身智能、具身降临（遥操作）、XR/VR、工业培训、动画制作、医疗康复、教育科研等多种场景

凭借出色的性能与成本优势，掌控具备未来消费级标准3D手部控制器的潜力，在规模化量产后，每双手套的制造成本约360元人民币

参考业内头部B端动捕手套厂商Manus Meta，当前年营收近1000万美元
参考Meta Quest头显当前市场规模，交互控制器硬件产品年市场容量预计超过10亿元人民币

核心特性

高自由度

从学术角度来看，手部的完整运动范围通常被分为 21 个自由度（不包括手腕）或 27 个自由度（包括手腕）。该解决方案真正涵盖了人类手部的完整运动范围，包括学术研究中常常简化或忽略的多个关节自由度，具有无与伦比的细节追踪能力。

高精度

手部动作捕捉是一个极为复杂的系统，目前没有标准的方法来定量测量误差，也没有现有的标准来与其他解决方案进行定量比较。通过定性分析渲染结果和真实动作的记录，可以明显看出该方案的精度水平远超任何已知的其它方案。

低延迟

过程 时间
计算 <1ms
无线（BLE 5.2 协议） 7.5-12.5ms
有线（USB 2.0 协议） <1ms

注：视频演示使用 USB 有线传输，显示硬件本身的最大刷新率为 60Hz。

自校准

理论上，用户在购买后无需进行任何手动校准。掌控可以通过自适应算法在正常使用过程中进行自校准。

该特性不同于大多数已知的穿戴式手部追踪设备，那些设备要求用户在每次使用前进行特定手势（例如握拳或张开手指），有时甚至需要在每次使用前进行。

捏合检测

基于专有硬件，该项技术能够在关键点（如两个指尖接触时）实现精确的单手捏合检测。基于此，它可以在毫米级精度下实现像按钮按下或滑动等功能的精确触发。

相比需要手部保持在摄像头视野内并且捏合动作相对较大的基于视觉的解决方案，该解决方案显著提升了用户体验，减少了用户的疲劳感。

高稳定性

  • 与 9 轴 IMU 方案相比 - 该方案不受 磁场 或 铁磁物质 的影响。
  • 与 电磁 方案相比 - 该方案不受环境中 金属物体 的干扰。
  • 与 拉伸、弯曲传感器 方案相比 - 该方案几乎避免了由 传感器老化 引起的 参数漂移。
  • 与 计算机视觉 方案相比 - 该方案显然不受 光线因素 或 视觉遮挡 等因素的影响。

易穿戴

只需两步：戴上手套并将魔术贴固定在手腕。

为适应不同的手部尺寸，提供几种标准的弹性手套尺码。手指的粗细不会影响适配性，手套旨在尽可能匹配不同的手指长度。

市场机会

具身智能与机器人团队

需求：可规模化采集的真实人手多模态数据（含与灵巧手同源的触觉传感器数据，用于生成面向灵巧手训练的高质量多模态数据集）；细粒度关节追踪与可靠标注。

购买方：企业/高校研究实验室、数据采集团队。

具身降临（遥操作）

应用场景：危险环境作业（危重病房、电力）、搜救作业等。

购买方：机器人公司、特种作业设备商。

XR/VR 厂商

需求：高精度、低延迟、稳定交互；标准化 SDK 与硬件接口，便于集成至头显与内容生态。

购买方：头显/控制器团队、系统集成伙伴、平台方 BD。

VR大空间

需求：稳定可靠的手部交互与捏合识别；现场部署可靠、维护简便。

购买方：VR主题乐园、沉浸式体验馆、线下文旅场馆。

工业培训与仿真

需求：自然手部操作与姿态还原；高可靠性、易维护，适配主流仿真/引擎。

购买方：仿真系统厂商、职业院校/培训机构。

动画与特效制作

需求：细节级手指动作捕捉；与 DCC/实时引擎（如 Unreal/Unity）无缝对接。

购买方：Mocap 工作室、影视/游戏内容公司。

医疗康复与科研

需求：高分辨率手部运动数据；设备易佩戴、可消毒复用；研究级导出格式。

购买方：康复设备商、医院/研究机构（科研用途）。

教育与创客生态

需求：低成本、可编程、可课程化；配套示例与教学资源。

购买方：高校/中学实验室、创客空间、研学机构。
